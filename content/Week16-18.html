Week16-18

week16  IPV6對外連線個人倉儲

1.在virtualbox網路設定將兩個ipv6選項打勾，ubuntu設定網路設為net work，開啟ubuntu

2.輸入密碼為kmol2020，開啟LXTermial，輸入 cd etc > cd apt >cd apt.conf.d > sudo vi proxy.conf

3.輸入密碼為kmol2020，按 i 編輯成 Acquire::http::Proxy:: "[2001:288:6004:17::69]:3128"; ，接著按 esc 並輸入 :wq 存檔結束

4.在apt下輸入 sudo apt update 更新apt，輸入 sudo apt install net-tools， cd etc > cd netplan > sudo vi 00-install-config.yaml，編輯內容如影片，接著按 esc 並輸入 :wq 存檔結束

5.在netplan 下輸入ping6 2001:288:6004:17::254測試，新增ipv4及ipv6埠號如影片，ipv6 的 guestip 要在 netplan下輸入 ifconfig 查詢，輸入 fd17 那行

6.進入tmp，輸入 git config -- global --add http.proxy [2001:288:6004:17::69]:3128，完成後輸入 git clone --recurse-submodules 個人倉儲網址

7.進入cd2020 > cmsimde，輸入sudo vi wsgi.py，將近端host網址更改為個人ipv6的gusetip(fd17...)，輸入 python3 wsgi.py 開啟近端

8.新增一個 ipv6埠號為9443，將ipv6的hostip改為  2001:288:6004:17:2020:2::72個人號碼

9.變更網路介面卡設定 ipv6位址為 2001:288:6004:17:2020:2::72個人號碼，長度64，閘道2001:288:6004:17::254，關閉proxy及防火牆，開起個人近端連線網址為 https://[2001:288:6004:17:2020:2::72個人號碼]:19443

影片: https://youtu.be/IltHEl-jKOw
