week11  利用ssh維護cd2020倉儲

1.將cd2020的.git資料夾裡的config拉進scite裡編輯，將原先url = https://github.com/個人github帳號/cd2020.git 前面上一個 #  ，重新創建   url = git@github.com:個人github帳號/cd2020.git 

2.將原先py373裡的putty放置新創建的py382資料夾下，將py373下home_mdecourse資料夾裡的.ssh資料夾複製到py382下

3.開啟putty.exe，load原先的github.com，如果是用學校電腦要去設定porxy為69，如果是用自己電腦就不需要設定proxy，打開ssh分支下的auth將金鑰路徑設定為cd2020倉儲下的金鑰位置

4.登陸putty，可以成功登陸後就可以試驗ssh有無成功

如果無法利用以上步驟直接設定ssh，可以參照http://mde.tw/wcmj2020/content/%E4%BB%A5%20SSH%20%E7%B6%AD%E8%AD%B7%E5%80%89%E5%84%B2.html 步驟重新設定一次新的ssh金鑰

week11  vrep虛擬主機設置

1.下載 virtualbox WINDOWS版，下載 ubuntu

2.開啟 virtualbox > 檔案 > 喜好設定 > 網路 > 新增一個nat網路，點選檔案 > 匯入應用裝置 > 匯入剛剛下載的 ubuntu

3.在匯入好的 ubuntu點選 設定 > 網路 > 僅限主機卡介面，完成後啟動

4.輸入密碼為kmol2020，點選 activities > 開啟LXTerminal > 輸入 xterm& > 輸入 ls > 輸入 cd coppeliasim4_rev4 > 輸入 ls > 輸入 cd CoppeliaSim_Edu_V4_0_0_Ubuntu18_04 > 輸入 ls

5.等它跑一段後輸入 ls -l > 輸入 ./coppeliaSim.sh 來開啟vrep

6.到file > open recent scene > sdc.ttt 檔即可

操作影片 : https://youtu.be/T2y29mgNzoM

week13  IPV4 Ubuntu對外連線設定

1.開啟 virtualbox ，喜好設定網路新增網路連接埠號，新增四個埠號，相關設定如影片操作，到ubuntu設定網路為NAT網路，開啟ubuntu

2.輸入密碼kmol2020，開啟LXTerm，輸入 inconfig，輸入sudo apt install net-tools，密碼為 kmol2020，輸入 cd tmp > ls ，輸入 ping 127.0.0.1 測試，這裡要按ctri+c不然會跑一堆出來不停

3.輸入cd cd2020pj1 > git pull，輸入 sudo vi wsgi.py 來編輯，要按 i 才能編輯，將近端改成10.0.2.4，接著按 esc 並輸入 :wq 存檔結束

4.輸入leo，選擇 open-Outline 開啟 cd2020pj1.leo，找到 @path static > @edit axuploade r下第30行的 allowExt，複製到 Nav搜尋，點選 @edit uploadform 下那一行 > 在gif後新增 'ttt' 後存檔

5.回到tmp下輸入 sudo vi oauth_scrum.txt，同樣按 i 編輯內容可以隨便打，同樣按esc並輸入 :wq 存檔結束，進到 cd2020pj1，輸入 python3 wsgi.py 啟動，如果無法啟動要先輸入 pip3 install pydrive 安裝老師新增的模組就可以了

6.網址為 https:/127.0.0.1:18443/alogin，帳號密碼為 admin，進入後可以新增ttt檔即完成 Ubuntu對外連線設定

影片 : https://youtu.be/lVg-58ChUi8

week14  IPV6 Ubuntu對外連線設定(要用學校電腦)

  1. 在virtualbox網路設定將兩個ipv6選項打勾，ubuntu設定網路設為net work，開啟ubuntu
  2. 輸入密碼為kmol2020，開啟LXTermial，輸入 cd etc > cd apt >cd apt.conf.d > sudo vi proxy.conf
  3. 輸入密碼為kmol2020，按 i 編輯如影片，接著按 esc 並輸入 :wq 存檔結束
  4. 輸入sudo apt > sudo apt install net-tools > cd etc > cd netplan > sudo vi 00-install-config.yaml，編輯內容如影片
  5. 輸入sudo netplane > ping6 2001:288:6004:17::254，在virtualbox添加ipv6的埠號如影片
  6. 輸入inconfig來查詢ip然後輸入到ipv6設定裡，如影片操作
  7. 進入cd2020pj1 > git config --global –add http.proxy[2001:288:6004:17::53]:3128
  8. 輸入git pull，進入tmp > sudo vi oauth_scrum.txt，輸入 sudo vi wsgi.py 來編輯，要按 i 才能編輯，將近端改成10.0.2.4，接著按 esc 並輸入 :wq 存檔結束
  9. 進到 cd2020pj1 > python3 wsgi.py 啟動，網址為https://[::1]:18443/alogin，開啟leo > 開啟 cd2020pj1.leo，找到 @path static > @edit axuploade 下第30行的 allowExt，複製到 Nav搜尋，點選 @edit uploadform 下那一行 > 在gif後新增 'ttt' 後存檔

影片 :　https://youtu.be/_WwFeXtjCtY
